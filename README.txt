Esta consiste en un sitio web de e-commerce que emplea: 

- un cart funcional: con el que el usuario puede seleccionar los productos que va a comprar.
- firebase: donde se guardan los datos de los productos
- cartContext: para las funciones del carrito
- rutas: para navegar entre las distintas páginas de productos, categorías, carrito, checkout, y demás
- itemDetail: para que el usuario tenga la posibilidad de elejir la cantidad de productos de su preferencia
- CheckOut: con el que el sitio generará una orden relacionada a los datos del usuario y a los productos en el carrito
- variables de entorno: para proteger la seguridad de las credenciales de firebase
- estilos css: para todos los componentes del sitio
